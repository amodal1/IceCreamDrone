IceCreamDrone

A project for collecting, creating, and curating units of behavior for autonomous creatures. In this case, flying drones.

View project onGitHub

Welcome to IceCreamDrone

Welcome to the IceCreamDrone project. The goal is to create, aggregate, and curate a system of parameterized behaviors for flying drones (both as individual agents and as various organized collections).

The project is maintained by Robert ( @amodal1 ) and Erica ( @ericacooksey ) Cooksey
